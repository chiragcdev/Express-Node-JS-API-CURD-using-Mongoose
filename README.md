# Express-Node-JS-API-CURD-using-Mongoose
Express-Node-JS-API-CRUD-using-Mongoose is a RESTful backend application built with Express and Node.js that performs Create, Read, Update, and Delete operations on MongoDB collections using Mongoose for schema-based data modeling and validation.

**1. Install Dependencies:-** npm install express --save

**Or manually:** npm install mongoose

**Enable ES Modules (Optional)** :- Add to package.json: "type": "module"

**Run the Server**:- nodemon index.js

**Server runs**:- http://localhost:3500
